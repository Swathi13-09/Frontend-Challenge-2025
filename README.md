## Project Overview

The Focus Timer allows users to:

* Set a timer for 15, 30, 45 minutes, or 1 hour
* Write their purpose before starting the timer
* See motivational messages after completing sessions
* Change theme colors (Black, Green, and Purple)
* Start, Pause, and Reset the timer easily
